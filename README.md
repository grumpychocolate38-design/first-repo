# first-repo
cursor-claude-codex
## Tools Installed
- Cursor IDE v3.6.21
- Claude Code extension
- Codex extension

## Steps Completed
1. Downloaded Cursor installer from cursor.sh
2. Ran CursorUserSetup-x64-3.6.21.exe
3. Windows Defender SmartScreen blocked the app — clicked "More info" → "Run anyway" to proceed
4. In Cursor setup dialog, selected additional tasks:
   - Create a desktop icon
   - Add "Open with Cursor" to Windows Explorer file context menu
   - Add to PATH (requires shell restart)
5. Completed Cursor installation
6. Opened Cursor and navigated to Extensions (Ctrl+Shift+X)
7. Searched for and installed "Claude Code" extension
8. Searched for and installed "Codex" extension
9. Created a public GitHub repository at github.com
10. Cloned the repository locally using `git clone https://github.com/[username]/[repo-name].git`
11. Opened the cloned folder in Cursor (File → Open Folder)
12. Created README.md file documenting the setup process

## Issues & Solutions
- **Windows Defender SmartScreen block**: The installer wasn't recognized by Microsoft's reputation system. Solved by clicking "More info" → "Run anyway"
- **Unfamiliar with Cursor setup**: Watched YouTube and read Claude tutorials to understand the installation steps and extension installation process
- **GitHub repository setup**: Used YouTube tutorial to learn how to create a repository, clone it locally, and push changes to GitHub
